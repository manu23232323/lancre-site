# L'Ancre - Landing Page

Site vitrine pour L'Ancre, agence de sprints digitaux locaux basée à Vaux-le-Milieu, Isère.

## Services proposés

- **Starter** (1 490€) : Local Demand Sprint - Optimisation Google Business Profile
- **Boost** (1 990€) : Starter + Réputation Propre (système avis)
- **Pro** (2 590€) : Écosystème complet (+ No-Show Killer)

## Déploiement

✅ **Site déployé automatiquement sur Netlify depuis GitHub**

Le site se redéploie automatiquement à chaque push sur la branche `main`.

### Netlify Forms

Le formulaire de contact utilise Netlify Forms pour la collecte des demandes.
Les soumissions sont visibles dans le dashboard Netlify : **Site settings > Forms**

## Structure

- `index.html` - Page unique avec toutes les sections
- `netlify.toml` - Configuration Netlify
